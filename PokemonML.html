    Pokemon Battle AI (WORK IN PROGRESS)

    This project uses a custom genetic algorithm and neural networks in order to create Pokemon battling AI inside a simulator. The battle simulator (PokemonEssentials by Maruno) was created in RPG Maker XP and coded in Ruby. Some modifications were made in the code by me. The genetic algorithm and neural networks were inspired by my research (sources listed in sources.txt) and written by me in Python using TensorFlow.
